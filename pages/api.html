API Reference

Programmatic access to all data. Free API key, 300 requests/minute.

MCP server — use this data inside Claude, Cursor & other AI tools

The ElkassabgiData MCP server exposes both of our libraries (this one and Econ Data Library) to any AI assistant that speaks the Model Context Protocol — search series, fetch data with citations, and run guided analyses directly in a conversation.

Connect (e.g. Claude → Settings → Connectors → Add custom connector):

https://elkassabgidata-mcp.elkassabgi.workers.dev/mcp?api_key=YOUR_API_KEY

Signed in? Your personal connect URL — with your key already inside, nothing to edit — appears here automatically. Sign in / register.

Use the same free API key as this site (browse/search tools work without one; data downloads require it). The key stays server-side — it is never echoed into your AI conversation.

The server also ships MCP resources (a data-honesty charter — survivorship, source-break and licensing caveats your AI relays with analyses — plus the variables dictionary) and guided prompts (analyze_econ_series, compare_countries, hf_event_study). Try: “Search ElkassabgiData for German inflation and plot it against US CPI since 2015.”

Getting started

Base URL: https://api.hfdatalibrary.com/v1

All requests require an API key passed as a header:

curl -H "X-API-Key: your-key-here" \
  https://api.hfdatalibrary.com/v1/symbols

API keys are free and expire every 30 days. Register here with your name, institution, and email.

Rate limit: 300 requests per minute per API key. Responses include X-RateLimit-Remaining and X-RateLimit-Reset headers.

Endpoints

GET /symbols

List all available tickers with metadata.

{
  "count": 1391,
  "symbols": [
    {
      "ticker": "AAPL",
      "name": "Apple Inc.",
      "start_date": "2002-12-30",
      "end_date": "2026-04-04",
      "bars_raw": 2847391,
      "bars_clean": 2846982,
      "quintile": 5,
      "source": "pitrading+iex"
    },
    ...
  ]
}
GET /symbols/{ticker}

Metadata for a single ticker: date range, bar counts, quality scores, data source.

GET /bars/{ticker}

Query 1-minute OHLCV bars for a ticker.

Parameters

ParamTypeDefaultDescription
startdateearliestStart date (YYYY-MM-DD)
enddatelatestEnd date (YYYY-MM-DD)
versionstringcleanData version: raw or clean
formatstringjsonResponse format: json, csv, or parquet
limitint10000Max rows per response (pagination)
offsetint0Row offset for pagination

GET /bars/{ticker}/daily

Daily OHLCV bars aggregated from intraday data. Same parameters as /bars/{ticker}.

GET /quality/{ticker}

Daily data quality report: gap rate, observed bars, longest gap, bars since last trade.

500Server error — please report to admin@hfdatalibrary.com
