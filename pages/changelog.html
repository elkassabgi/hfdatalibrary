Changelog

Every data update, versioned and documented.

Data is updated daily via automated pipeline. Each update adds the previous trading day's data for all 1,391 tickers across both cleaning versions.

v1.0 — April 9, 2026 (Initial release)

  • 1,391 U.S. equities and ETFs
  • Two cleaning versions: Raw and Clean
  • Coverage: January 1991 through April 2026 (majority from December 2002)
  • Primary source: PiTrading consolidated tape (pre-March 2022)
  • Secondary source: IEX Exchange HIST (post-March 2022)
  • Nine-step cleaning pipeline
  • 25 pre-computed academic variables per ticker per day
  • Parquet file format
  • REST API with JSON, CSV, and parquet response formats
